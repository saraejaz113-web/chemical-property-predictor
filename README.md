# Chemical Property Predictor

This project uses simple python concepts and determines whether a chemical solution is acidic, basic or neutral.It uses decision-making logic to classify the chemical solutions.The rules used are based on basic chemistry concepts.

## Features
- pH value of provided solution is taken as input.
- identifies chemical solutions as acidic, basic or neutral.

## Learning Outcome
Through this program, I learned how to use simple python concepts to evaluate the chemical nature of a solution.
